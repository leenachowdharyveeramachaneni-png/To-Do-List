# To-Do List
A full stack task manager built using HTML, CSS, JavaScript, Node.js, and MySQL.

[Live Demo](https://to-do-list-3-i3ma.onrender.com)

## Features
* User login and registration
* Add and delete tasks
* Mark tasks as completed
* Data stored in MySQL

## How to Run
1. Run backend:
   node server/server.js

2. Open login.html using Live Server
